Motioneye
=========

A web frontend for the motion daemon, written in Python.

The entry point script is :mod:`motioneye.meyectl`

Main arguments:
    :command: ``startserver`` | ``stopserver`` | ``sendmail`` | ``webhook`` | ``shell``
